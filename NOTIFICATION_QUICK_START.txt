✅ SETUP IN 3 EASY STEPS:

STEP 1: Create Database Table
────────────────────────────────
Open phpMyAdmin → Select 'mathease' database → SQL tab
Run this file: php/setup-notifications-table.sql

OR use command line:
mysql -u root -p mathease < php/setup-notifications-table.sql


STEP 2: Test the System
────────────────────────────────
1. Login to teacher dashboard
2. Visit: http://localhost/MathEase/php/test-notifications.php
3. This creates sample notifications


STEP 3: Check Dashboard
────────────────────────────────
1. Look for bell icon (top right)
2. See red badge with number
3. Click bell to view notifications
4. Click notification to mark as read

═══════════════════════════════════════════════════════════════
  🔍 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: No bell icon showing?
Solution: Clear cache (Ctrl+F5) and refresh dashboard

Problem: Badge showing 0 even with notifications?
Solution: Notifications might be marked as read
         Run test script to create new ones

Problem: "Table doesn't exist" error?
Solution: Run setup-notifications-table.sql first

Problem: Notifications not auto-refreshing?
Solution: Check browser console (F12) for JavaScript errors


═══════════════════════════════════════════════════════════════
  💡 USAGE EXAMPLES
═══════════════════════════════════════════════════════════════

Add to any PHP file to create notifications:

require_once 'create-notification.php';

// When student joins class
notifyNewEnrollment($teacher_id, $class_id, $student_name, $class_name);

// When student submits quiz
notifyQuizSubmission($teacher_id, $class_id, $student_name, $quiz_name, $class_name);

// Custom notification
createTeacherNotification(
    $teacher_id,
    'class_update',
    'Custom Title',
    'Your custom message here',
    $class_id
);
