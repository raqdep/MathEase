MathEase System - Complete Data Flow Diagram (DFD) Files - Levels 0-3
================================================================================

OVERVIEW:
---------
This directory contains complete Data Flow Diagrams (DFD) for the MathEase 
Learning Platform across all levels (0-3). DFDs visualize how data flows 
through the system, showing processes, data stores, external entities, and 
their relationships.

FILES:
------
1. MathEase_DFD_Level0.xml     - Context Diagram (System Overview)
2. MathEase_DFD_Level1.xml     - System Decomposition (9 Major Processes)
3. MathEase_DFD_Level2.xml     - Process 3.0 Decomposition (Class Management)
4. MathEase_DFD_Level3.xml     - Process 3.2 Decomposition (Join Class Request)
5. DFD_Complete_Documentation.md - Complete documentation for all levels

HOW TO VIEW:
-----------
1. Open Draw.io (diagrams.net) - https://app.diagrams.net/
2. Click "Open Existing Diagram"
3. Upload any XML file
4. The diagram will load and be fully editable

You can also:
- Download Draw.io desktop app for offline editing
- Export diagrams as PNG, PDF, or SVG

DFD LEVELS:
-----------
Level 0: Shows the system as one process with external entities
Level 1: Breaks down into 9 major processes and 16 data stores
Level 2: Decomposes Process 3.0 (Class Management) into 5 sub-processes
Level 3: Decomposes Process 3.2 (Join Class Request) into 5 detailed sub-processes

DFD HIERARCHY:
--------------
Level 0: MathEase Learning Platform
    └── Level 1: System Decomposition (9 processes)
        └── Level 2: Process 3.0 Class Management (5 sub-processes)
            └── Level 3: Process 3.2 Join Class Request (5 detailed sub-processes)

KEY COMPONENTS:
--------------
External Entities:
- Student, Teacher, Admin, Email System

Major Processes (Level 1):
1. User Authentication
2. Registration & Email Verification
3. Class Management
4. Lesson Management
5. Quiz Management
6. Progress Tracking
7. Notification Management
8. Admin Management
9. Badge & Achievement System

Data Stores:
- Users, Teachers, Classes, Enrollments, Topics, Lessons, Quizzes,
  Progress, Notifications, Badges, and more (16 total)

For detailed information, see DFD_Complete_Documentation.md
