Solution 2: Fix Gmail App Password

Make sure your Gmail App Password is correct:

  1. Go to Google Account Security
  2. Enable 2-Step Verification if not already enabled
  3. Go to App passwords
  4. Generate a new App Password for "Mail"
  5. Update the password in php/config.php

Solution 3: Use Different SMTP Settings

Try alternative Gmail SMTP settings:

SMTP Host: smtp.gmail.com
SMTP Port: 465 (instead of 587)
Encryption: SSL (instead of TLS)

Solution 4: Use Email Service Provider

  • SendGrid: Free tier with 100 emails/day
  • Mailgun: Free tier with 5,000 emails/month
  • Amazon SES: Very cheap, reliable
