What are Rational Functions?

Definition

A rational function is a function that can be expressed as the ratio of two polynomial functions. It has the general form:

f(x) = P(x) / Q(x)

where P(x) and Q(x) are polynomials and Q(x) ≠ 0

Key Characteristics

Domain Restrictions

The domain includes all real numbers except where Q(x) = 0. These excluded values create vertical asymptotes.

What this means

Any x that makes the denominator zero is not allowed, because division by zero is undefined.

Asymptotic Behavior

Rational functions approach horizontal asymptotes as x approaches ±∞, depending on the degrees of P(x) and Q(x).

Tip

Compare the highest powers of x in the numerator and denominator to quickly identify the horizontal asymptote.

Examples of Rational Functions

f(x) = 1/x
Simple reciprocal function
f(x) = (x² - 1)/(x + 2)
Quadratic over linear
f(x) = (2x + 3)/(x² - 4)
Linear over quadratic

Worked Examples

Example 1: Identifying Rational Functions

Determine which of the following are rational functions:

a) f(x) = 3x² + 2x - 1 Not Rational
b) f(x) = (x + 1)/(x - 2) Rational
c) f(x) = √x Not Rational

Example 2: Finding Domain

Find the domain of f(x) = (x + 3)/(x² - 9)

Solution:

Set the denominator equal to zero: x² - 9 = 0

Factor: (x + 3)(x - 3) = 0

Solve: x = -3 or x = 3

Domain: All real numbers except x = -3 and x = 3

Domain: (-∞, -3) ∪ (-3, 3) ∪ (3, ∞)

Graphing Rational Functions

Step-by-Step Graphing Process

1 Find Vertical Asymptotes

Set the denominator equal to zero and solve for x. These values create vertical lines that the graph approaches but never crosses.

2 Find Horizontal Asymptotes

Compare the degrees of numerator and denominator polynomials to determine horizontal asymptotes.

3 Find Intercepts

Calculate x-intercepts (where f(x) = 0) and y-intercepts (where x = 0).

4 Plot Key Points and Sketch

Plot intercepts and additional points, then sketch the curve following the asymptotes.

Types of Asymptotes

Vertical Asymptotes

Occur where the denominator equals zero.

x = a (where Q(a) = 0)

Horizontal Asymptotes

Depend on degrees of P(x) and Q(x).

y = 0, y = a, or none

Asymptote Analysis

Vertical Asymptotes

Method:
  1. Set the denominator equal to zero: Q(x) = 0
  2. Solve for x to find the vertical asymptote(s)
  3. Check if the numerator is also zero at these points (holes vs. asymptotes)

Example: For f(x) = (x+1)/(x-2), vertical asymptote at x = 2

Horizontal Asymptotes

Case 1: Degree of P(x) < Degree of Q(x)

Horizontal asymptote at y = 0

Example: f(x) = (x+1)/(x²-4) → y = 0

Case 2: Degree of P(x) = Degree of Q(x)

Horizontal asymptote at y = (leading coefficient of P)/(leading coefficient of Q)

Example: f(x) = (2x+1)/(3x-2) → y = 2/3

Case 3: Degree of P(x) > Degree of Q(x)

No horizontal asymptote (may have oblique asymptote)

Example: f(x) = (x²+1)/(x-1) → oblique asymptote

Finding Intercepts

X-intercepts

Set f(x) = 0, which means P(x) = 0

Solve: P(x) = 0

Y-intercept

Set x = 0 and evaluate f(0)

Calculate: f(0) = P(0)/Q(0)

Applications of Rational Functions

Physics Applications

Electrical Resistance

When resistors are connected in parallel, the total resistance is given by:

1/R_total = 1/R₁ + 1/R₂ + 1/R₃

This creates a rational function relationship between individual resistances and total resistance.

Optics - Lens Equation

The lens equation relates object distance, image distance, and focal length:

1/f = 1/d₀ + 1/dᵢ

Where f is focal length, d₀ is object distance, and dᵢ is image distance.

Economics Applications

Average Cost Function

The average cost per unit is a rational function:

AC(x) = C(x)/x

Where C(x) is the total cost function and x is the number of units produced.

Market Share

Market share can be modeled as a rational function of advertising expenditure:

S(a) = ka/(a + b)

Where a is advertising expenditure, k and b are constants.

Engineering Applications

Control Systems

Transfer functions in control systems are often rational functions:

G(s) = N(s)/D(s)

Where N(s) and D(s) are polynomials in the complex variable s.

Vehicle Performance

Fuel efficiency as a function of speed can be modeled as:

E(v) = av/(v² + bv + c)

Where v is velocity and a, b, c are vehicle-specific constants.
