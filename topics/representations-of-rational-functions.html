Topic 1: Understanding Rational Functions

Learn the fundamental concepts of rational functions, their definitions, and how to identify them in various forms.

00:00 Study Time

Learning Objectives

  • Define rational functions and identify their components
  • Understand the domain and range of rational functions
  • Recognize rational functions in different representations
  • Apply rational functions to solve real-world problems

What are Rational Functions?

Definition

A rational function is a function that can be expressed as the ratio of two polynomial functions. It has the general form:

f(x) = P(x) / Q(x)

where P(x) and Q(x) are polynomials and Q(x) ≠ 0

Key Characteristics

Domain Restrictions

The domain includes all real numbers except where Q(x) = 0. These excluded values create vertical asymptotes.

What this means

Any x that makes the denominator zero is not allowed, because division by zero is undefined.

Asymptotic Behavior

Rational functions approach horizontal asymptotes as x approaches ±∞, depending on the degrees of P(x) and Q(x).

Tip

Compare the highest powers of x in the numerator and denominator to quickly identify the horizontal asymptote.

Examples of Rational Functions

f(x) = 1/x
Simple reciprocal

Domain: All real numbers except x = 0

Key feature: Vertical asymptote at x = 0, horizontal asymptote at y = 0

f(x) = (x² - 1)/(x + 2)
Quadratic/Linear

Domain: All real numbers except x = -2

Key feature: Can be simplified to f(x) = x - 2 + 3/(x + 2) (with hole at x = -1)

f(x) = (2x + 3)/(x² - 4)
Linear/Quadratic

Domain: All real numbers except x = 2 and x = -2

Key feature: Two vertical asymptotes, horizontal asymptote at y = 0

Graphs provide visual insight into function behavior, showing asymptotes, intercepts, and overall shape.

Key elements to identify:

Common Patterns and Simplifications

Pattern 1: Factoring

Many rational functions can be simplified by factoring:

f(x) = (x² - 4)/(x - 2)

= (x - 2)(x + 2)/(x - 2)

= x + 2, where x ≠ 2

Note: There's a hole at x = 2, not an asymptote!

Pattern 2: Long Division

When numerator degree ≥ denominator degree, use long division:

f(x) = (x² + 3x + 2)/(x + 1)

= x + 2 + 0/(x + 1)

= x + 2

Worked Examples

Example 1: Identifying Rational Functions

Determine which of the following are rational functions:

a) f(x) = 3x² + 2x - 1 Not Rational
b) f(x) = (x + 1)/(x - 2) Rational
c) f(x) = √x Not Rational

Example 2: Finding Domain

Find the domain of f(x) = (x + 3)/(x² - 9)

Step-by-Step Solution:

1

Set the denominator equal to zero: x² - 9 = 0

2

Factor using difference of squares: (x + 3)(x - 3) = 0

3

Apply zero product property: x + 3 = 0 or x - 3 = 0

4

Solve: x = -3 or x = 3

Final Answer:

Domain: All real numbers except x = -3 and x = 3

Domain: (-∞, -3) ∪ (-3, 3) ∪ (3, ∞)

These excluded values create vertical asymptotes at x = -3 and x = 3

Example 3: Simplifying Rational Functions

Simplify f(x) = (x² - 4)/(x - 2) and identify any holes or asymptotes.

Solution:

f(x) = (x² - 4)/(x - 2)

= (x - 2)(x + 2)/(x - 2)

= x + 2, where x ≠ 2

Important: Even though we canceled (x - 2), x = 2 is still excluded from the domain. This creates a hole (not an asymptote) at x = 2.

Hole location: (2, 4) - found by evaluating x + 2 at x = 2

Graphing Rational Functions

Step-by-Step Graphing Process

1 Find Vertical Asymptotes

Set the denominator equal to zero and solve for x. These values create vertical lines that the graph approaches but never crosses.

2 Find Horizontal Asymptotes

Compare the degrees of numerator and denominator polynomials to determine horizontal asymptotes.

3 Find Intercepts

Calculate x-intercepts (where f(x) = 0) and y-intercepts (where x = 0).

4 Plot Key Points and Sketch

Plot intercepts and additional points, then sketch the curve following the asymptotes.

Types of Asymptotes

Vertical Asymptotes

Occur where the denominator equals zero (and numerator doesn't).

x = a (where Q(a) = 0)

Example: f(x) = 1/(x - 2) has vertical asymptote at x = 2

Behavior: As x approaches 2, f(x) approaches ±∞

Horizontal Asymptotes

Depend on degrees of P(x) and Q(x).

y = 0, y = a, or none

Rule: Compare highest degree terms

If deg(P) < deg(Q): y = 0

If deg(P) = deg(Q): y = leading coefficients ratio

If deg(P) > deg(Q): No horizontal asymptote

Complete Graphing Example

Example: Graph f(x) = (x + 1)/(x - 2)

Step 1: Find Vertical Asymptote

Set denominator = 0: x - 2 = 0 → x = 2

Vertical asymptote: x = 2

Step 2: Find Horizontal Asymptote

Degrees are equal (both degree 1), so y = leading coefficient ratio

Horizontal asymptote: y = 1/1 = 1

Step 3: Find Intercepts

X-intercept: Set numerator = 0 → x + 1 = 0 → x = -1

Y-intercept: f(0) = (0+1)/(0-2) = 1/(-2) = -0.5

Intercepts: (-1, 0) and (0, -0.5)

Step 4: Plot and Sketch

Plot intercepts, draw asymptotes as dashed lines, then sketch the curve approaching asymptotes.

Domain: (-∞, 2) ∪ (2, ∞)

Understanding Behavior Near Asymptotes

Rational functions exhibit predictable behavior as they approach asymptotes:

Near Vertical Asymptotes
  • • Function values approach ±∞
  • • Sign depends on approaching from left or right
  • • Graph never crosses the vertical line
Near Horizontal Asymptotes
  • • Function values approach the horizontal line
  • • Graph may cross horizontal asymptote
  • • Behavior as x → ±∞

Topic 3: Analyzing Asymptotes and Intercepts

Deep dive into the analysis of asymptotes and intercepts, understanding their mathematical significance and graphical implications.

00:00 Study Time

Learning Objectives (MELC: M11GM-Ib-3)

  • Calculate vertical asymptotes from denominator zeros
  • Determine horizontal asymptotes using degree analysis
  • Find x-intercepts by solving numerator equal to zero
  • Analyze behavior near asymptotes and at infinity

Asymptote Analysis

Vertical Asymptotes

Method:
  1. Set the denominator equal to zero: Q(x) = 0
  2. Solve for x to find the vertical asymptote(s)
  3. Check if the numerator is also zero at these points (holes vs. asymptotes)

Example: For f(x) = (x+1)/(x-2), vertical asymptote at x = 2

Horizontal Asymptotes

Case 1: Degree of P(x) < Degree of Q(x)

Horizontal asymptote at y = 0

Example: f(x) = (x+1)/(x²-4) → y = 0

Case 2: Degree of P(x) = Degree of Q(x)

Horizontal asymptote at y = (leading coefficient of P)/(leading coefficient of Q)

Example: f(x) = (2x+1)/(3x-2) → y = 2/3

Case 3: Degree of P(x) > Degree of Q(x)

No horizontal asymptote (may have oblique asymptote)

Example: f(x) = (x²+1)/(x-1) → oblique asymptote

Finding Intercepts

X-intercepts

Set f(x) = 0, which means P(x) = 0 (since denominator cannot be zero)

Solve: P(x) = 0

Important: Check that solutions don't make denominator zero (would be holes, not intercepts)

Example:

f(x) = (x² - 1)/(x + 2)

Set x² - 1 = 0 → x = ±1

X-intercepts: (-1, 0) and (1, 0)

Y-intercept

Set x = 0 and evaluate f(0). If Q(0) = 0, there is no y-intercept.

Calculate: f(0) = P(0)/Q(0)

Note: Y-intercept exists only if Q(0) ≠ 0

Example:

f(x) = (x + 1)/(x - 2)

f(0) = (0 + 1)/(0 - 2) = 1/(-2) = -0.5

Y-intercept: (0, -0.5)

Complete Analysis Example

Example: Analyze f(x) = (2x² + 3x - 1)/(x² - 4)

Vertical Asymptotes:

Set denominator = 0: x² - 4 = 0

Factor: (x - 2)(x + 2) = 0

Vertical asymptotes: x = 2 and x = -2

Horizontal Asymptote:

Degrees are equal (both degree 2)

Horizontal asymptote = leading coefficient ratio

Horizontal asymptote: y = 2/1 = 2

Intercepts:

X-intercepts: Solve 2x² + 3x - 1 = 0 using quadratic formula

x = (-3 ± √17)/4

Y-intercept: f(0) = -1/(-4) = 0.25 → (0, 0.25)

Holes vs. Vertical Asymptotes

It's crucial to distinguish between holes and vertical asymptotes:

Holes

Occur when both numerator and denominator are zero at the same x-value.

Example: f(x) = (x-2)/(x-2)

Hole at x = 2

The function can be simplified, but x = 2 is still excluded.

Vertical Asymptotes

Occur when only the denominator is zero (numerator is not zero).

Example: f(x) = 1/(x-2)

Vertical asymptote at x = 2

The function approaches ±∞ as x approaches 2.

Applications of Rational Functions

Physics Applications

Electrical Resistance

When resistors are connected in parallel, the total resistance is given by:

1/R_total = 1/R₁ + 1/R₂ + 1/R₃

This creates a rational function relationship between individual resistances and total resistance.

Optics - Lens Equation

The lens equation relates object distance, image distance, and focal length:

1/f = 1/d₀ + 1/dᵢ

Where f is focal length, d₀ is object distance, and dᵢ is image distance.

Economics Applications

Average Cost Function

The average cost per unit is a rational function:

AC(x) = C(x)/x

Where C(x) is the total cost function and x is the number of units produced.

Market Share

Market share can be modeled as a rational function of advertising expenditure:

S(a) = ka/(a + b)

Where a is advertising expenditure, k and b are constants.

Engineering Applications

Control Systems

Transfer functions in control systems are often rational functions:

G(s) = N(s)/D(s)

Where N(s) and D(s) are polynomials in the complex variable s.

Application: Used to model system response, stability analysis, and controller design

Vehicle Performance

Fuel efficiency as a function of speed can be modeled as:

E(v) = av/(v² + bv + c)

Where v is velocity and a, b, c are vehicle-specific constants.

Domain restriction: v > 0 (speed must be positive)

Interpretation: Efficiency peaks at optimal speed, decreases at very low or very high speeds

Step-by-Step Application Problem

Example: Average Cost Problem

A company's total cost function is C(x) = 1000 + 5x, where x is the number of units produced. Find the average cost function and determine the average cost when producing 100 units.

Step 1: Identify the Average Cost Function

AC(x) = C(x)/x = (1000 + 5x)/x

Step 2: Simplify the Function

AC(x) = 1000/x + 5x/x = 1000/x + 5

Step 3: Evaluate at x = 100

AC(100) = 1000/100 + 5 = 10 + 5 = 15

Answer: The average cost per unit is $15 when producing 100 units.

Step 4: Domain Considerations

Domain: x > 0 (cannot produce negative or zero units)

Interpretation: As production increases, the fixed cost (1000) is spread over more units, reducing average cost.

Common Mistakes in Real-World Applications

Mistake 1: Ignoring Domain Restrictions

Error: Using negative values for quantities that must be positive (e.g., time, distance, production)

Solution: Always check the context and restrict the domain appropriately

Mistake 2: Not Simplifying Before Evaluating

Error: Directly substituting values into complex rational expressions

Solution: Simplify first, then evaluate for better accuracy

Mistake 3: Misinterpreting Asymptotes

Error: Thinking horizontal asymptotes represent maximum/minimum values

Solution: Asymptotes show limiting behavior, not actual maximums/minimums
