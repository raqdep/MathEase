Understanding Rational Functions

What is a Rational Function?

A rational function is a function that can be expressed as the ratio of two polynomials.

General Form:

f(x) = P(x)/Q(x)

Where P(x) and Q(x) are polynomials, Q(x) ≠ 0

Important Restrictions

Rational functions have restrictions on their domain:

! The denominator cannot be zero
! Domain excludes values that make Q(x) = 0
! These create vertical asymptotes

Examples of Rational Functions

Simple Rational Function

f(x) = 1/x

Domain: x ≠ 0

Range: y ≠ 0

Complex Rational Function

f(x) = (x² - 1)/(x - 2)

Domain: x ≠ 2

Range: All real numbers except y = 1

Enhanced Step-by-Step Domain Finding

Example: Find the domain of f(x) = (x + 3)/(x² - 4)
1
Identify the restriction

Rational functions are undefined where the denominator equals zero.

Set denominator = 0

x² - 4 = 0

2
Factor the denominator

Use the difference of squares formula: a² - b² = (a - b)(a + b)

x² - 4 = x² - 2²

(x - 2)(x + 2) = 0

3
Solve for x

Use the zero product property: if ab = 0, then a = 0 or b = 0

x - 2 = 0 or x + 2 = 0

x = 2 or x = -2

4
Write the domain

The domain includes all real numbers except the values that make the denominator zero.

Domain: {x | x ≠ 2, x ≠ -2}

or in interval notation: (-∞, -2) ∪ (-2, 2) ∪ (2, ∞)

Graphing Rational Functions: Complete Guide

Vertical Asymptotes

Vertical asymptotes occur where the denominator equals zero (but the numerator doesn't).

Finding Vertical Asymptotes

1. Set denominator = 0

2. Solve for x

3. Check if numerator ≠ 0 at these points

Example: f(x) = 1/(x - 3)

Vertical asymptote: x = 3

The function approaches ±∞ as x approaches 3

Horizontal Asymptotes

Horizontal asymptotes describe the behavior as x approaches ±∞.

Rules for Horizontal Asymptotes

• If degree of P(x) < degree of Q(x): y = 0

• If degree of P(x) = degree of Q(x): y = leading coefficients ratio

• If degree of P(x) > degree of Q(x): No horizontal asymptote

Example: f(x) = (2x + 1)/(x - 3)

Horizontal asymptote: y = 2

Both numerator and denominator have degree 1

Step-by-Step Graphing Process

Step 1: Find Domain

Identify values that make the denominator zero

Step 2: Find Asymptotes

Vertical: denominator = 0, Horizontal: compare degrees

Step 3: Find Intercepts

x-intercepts: numerator = 0, y-intercept: f(0)

Step 4: Plot Points

Choose test points and sketch the graph

Solving Rational Equations: Complete Guide

What are Extraneous Solutions?

Extraneous solutions are solutions that appear to work but don't satisfy the original equation.

Why do they occur?

• Multiplying by denominators can introduce false solutions

• Solutions that make any denominator zero are extraneous

• Always check solutions in the original equation!

Solution Process

Follow these steps to solve rational equations correctly:

Step 1: Find LCD

Find the least common denominator of all fractions

Step 2: Clear Fractions

Multiply both sides by the LCD

Step 3: Solve

Solve the resulting polynomial equation

Step 4: Check

Verify solutions in the original equation

Step-by-Step Examples

Example 1: Solve 1/x + 1/(x+2) = 5/12
1 Find LCD: x(x+2)
LCD = x(x+2)
2 Multiply by LCD
x(x+2)[1/x + 1/(x+2)] = x(x+2)[5/12]
3 Simplify
(x+2) + x = 5x(x+2)/12
2x + 2 = 5x² + 10x
4 Solve quadratic
5x² + 8x - 24 = 0
x = 1.2 or x = -4
5 Check solutions
Both solutions work!

Lesson 4: Rational Inequalities

Learn how to solve rational inequalities using sign analysis and number line methods.

Learning Objectives (MELC: M11GM-Ib-4)

  • Solve rational inequalities
  • Use sign analysis and number lines
  • Express solutions in interval notation

Solving Rational Inequalities: Complete Guide

Sign Analysis Method

Use sign analysis to determine where the rational function is positive or negative.

Step 1: Find Critical Points

• Zeros of numerator (where function = 0)

• Zeros of denominator (vertical asymptotes)

Step 2: Create Number Line

• Plot critical points on number line

• Test intervals between critical points

Interval Notation

Express solutions using interval notation for clear communication.

(a, b) - Open interval

All numbers between a and b, not including endpoints

[a, b] - Closed interval

All numbers between a and b, including endpoints

(-∞, a) ∪ (b, ∞) - Union

All numbers less than a or greater than b

Step-by-Step Examples

Example: Solve (x - 1)/(x + 2) ≥ 0
1 Find critical points
x = 1 (numerator zero), x = -2 (denominator zero)
2 Create number line
(-∞, -2) | (-2, 1) | (1, ∞)
3 Test intervals
x = -3: (-)/(-) = +
x = 0: (-)/(+) = -
x = 2: (+)/(+) = +
4 Write solution
(-∞, -2) ∪ [1, ∞)
