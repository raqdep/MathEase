Real-World Function Applications

Taxi Fare Problem

A taxi company charges ₱40 for the first kilometer and ₱15 for each additional kilometer. This is a common pricing structure in the Philippines.

Function Model:

f(x) = 40 + 15(x - 1) for x > 1

Where x = distance in kilometers

Step-by-Step Solution:

Example: For a 5km ride

• First km: ₱40

• Additional 4 km: 4 × ₱15 = ₱60

• Total: ₱40 + ₱60 = ₱100

Temperature Conversion

Convert Celsius to Fahrenheit using the formula F = (9/5)C + 32. This is essential for international travel and scientific work.

Function Model:

F(C) = (9/5)C + 32

Where C = temperature in Celsius

Step-by-Step Solution:

Example: Convert 25°C to Fahrenheit

• F(25) = (9/5)(25) + 32

• F(25) = 45 + 32 = 77°F

• So 25°C = 77°F

Piecewise Function: Electricity Bill

An electricity company charges different rates based on consumption. This tiered pricing system encourages energy conservation.

Piecewise Function:

f(x) = {

8x, if 0 ≤ x ≤ 100

800 + 12(x - 100), if x > 100

}

Where x = kilowatt-hours consumed

Understanding the Function:

For 0-100 kWh: ₱8 per kWh (basic rate)

For over 100 kWh: ₱800 + ₱12 per additional kWh

Example: 150 kWh consumption

• First 100 kWh: 100 × ₱8 = ₱800

• Additional 50 kWh: 50 × ₱12 = ₱600

• Total: ₱800 + ₱600 = ₱1,400

Mobile Phone Plan

A mobile plan charges ₱500 monthly with ₱2 per minute for calls beyond the included 100 minutes.

Function Model:

f(x) = 500 + 2(x - 100) for x > 100

Where x = total minutes used

Shipping Cost

A courier service charges ₱50 base fee plus ₱0.50 per kilogram for domestic shipping.

Function Model:

f(x) = 50 + 0.5x

Where x = weight in kilograms

Lesson 2: Business & Economics Applications

Apply functions to solve business problems including profit, cost, revenue, and economic models.

Learning Objectives (MELC: M11GM-Ia-2)

  • Model business scenarios using functions
  • Calculate profit, cost, and revenue functions
  • Solve optimization problems

Business Function Models

Revenue Function

R(x) = p × x

Where p = price per unit, x = quantity sold

Example: Selling phones at ₱15,000 each
R(x) = 15,000x

Cost Function

C(x) = Fixed Cost + Variable Cost

C(x) = FC + VC(x)

Example: FC = ₱50,000, VC = ₱8,000 per unit
C(x) = 50,000 + 8,000x

Profit Function

P(x) = R(x) - C(x)

Profit = Revenue - Cost

Example: P(x) = 15,000x - (50,000 + 8,000x)
P(x) = 7,000x - 50,000

Break-Even Analysis

The break-even point is where total revenue equals total cost, meaning no profit or loss. This is crucial for business planning and decision-making.

Step-by-Step Solution:

1. Set R(x) = C(x) (Revenue = Cost)

2. 15,000x = 50,000 + 8,000x

3. 15,000x - 8,000x = 50,000

4. 7,000x = 50,000

5. x = 50,000 ÷ 7,000 ≈ 7.14 units

Break-even point: 8 units (round up)

Business Interpretation:

Below 8 units: Company loses money (costs exceed revenue)

At 8 units: Company breaks even (no profit, no loss)

Above 8 units: Company makes profit (revenue exceeds costs)

Example: At 10 units: Revenue = ₱150,000, Cost = ₱130,000, Profit = ₱20,000

Retail Pricing Strategy

A store offers bulk discounts: ₱100 per item for 1-10 items, ₱90 per item for 11-50 items, ₱80 per item for 50+ items.

Piecewise Function:

P(x) = {

900 + 90(x-10), if 11 ≤ x ≤ 50

4500 + 80(x-50), if x > 50

}

Credit Card Interest

A credit card charges 2% monthly interest on outstanding balances, with a minimum payment of ₱500 or 5% of balance.

Interest Function:

Where b = outstanding balance

Lesson 3: Science & Technology Applications

Explore how functions model scientific phenomena, population growth, and technological applications.

Learning Objectives (MELC: M11GM-Ia-3)

  • Model scientific phenomena using functions
  • Apply exponential and logarithmic functions
  • Solve problems involving growth and decay

Scientific Function Models

Population Growth

Model population growth using exponential functions.

Exponential Growth Model:

P(t) = P₀ × e^(rt)

Where P₀ = initial population, r = growth rate, t = time

Example: A city with 100,000 people growing at 2% annually
P(t) = 100,000 × e^(0.02t)

Radioactive Decay

Model radioactive decay using exponential decay functions.

Exponential Decay Model:

N(t) = N₀ × e^(-λt)

Where N₀ = initial amount, λ = decay constant, t = time

Example: Carbon-14 with half-life of 5,730 years
N(t) = N₀ × e^(-0.000121t)

Newton's Law of Cooling

This law describes how the temperature of an object changes when it's placed in a different temperature environment. It's used in food safety, engineering, and meteorology.

Cooling Function:

T(t) = Tₐ + (T₀ - Tₐ)e^(-kt)

Where Tₐ = ambient temperature, T₀ = initial temperature, k = cooling constant

Real-World Application:

Example: Hot coffee (100°C) in room (20°C)

• T(t) = 20 + (100-20)e^(-0.1t)

• T(t) = 20 + 80e^(-0.1t)

• After 1 hour: T(1) = 20 + 80e^(-0.1) ≈ 52°C

Projectile Motion

This models the height of a projectile (like a ball, arrow, or rocket) over time, considering gravity's effect. Essential in physics, engineering, and sports.

Height Function:

h(t) = h₀ + v₀t - ½gt²

Where h₀ = initial height, v₀ = initial velocity, g = gravity (9.8 m/s²)

Real-World Application:

Example: Ball thrown upward from 2m at 15 m/s

• h(t) = 2 + 15t - 4.9t²

• Maximum height: t = 15/9.8 ≈ 1.53 seconds

• h(1.53) = 2 + 15(1.53) - 4.9(1.53)² ≈ 13.5m

Heart Rate During Exercise

A person's heart rate increases during exercise and decreases after stopping, following an exponential model.

Heart Rate Function:

H(t) = 60 + 40(1 - e^(-0.5t))

Where t = time in minutes, H = beats per minute

Plant Growth Model

Plant height growth follows a logistic function, growing rapidly at first then slowing as it approaches maximum height.

Growth Function:

G(t) = 200/(1 + 19e^(-0.1t))

Where t = time in days, G = height in cm

Complex Real-Life Problems

Problem 1: Manufacturing Optimization

A company manufactures two products. Product A costs ₱50 to make and sells for ₱80. Product B costs ₱30 to make and sells for ₱60. The company can produce at most 100 units of A and 150 units of B per month. How many of each should they produce to maximize profit?

Solution Steps:

1. Define variables: x = units of A, y = units of B

2. Profit function: P(x,y) = 30x + 30y

3. Constraints: x ≤ 100, y ≤ 150, x ≥ 0, y ≥ 0

4. Maximum profit at corner points

5. Solution: x = 100, y = 150, P = ₱7,500

Problem 2: Environmental Impact

A city's air quality is modeled by A(t) = 50 + 30sin(πt/12) + 10t, where A is the air quality index and t is time in months. When will the air quality first exceed 100?

Solution Steps:

1. Set A(t) = 100

2. 50 + 30sin(πt/12) + 10t = 100

3. 30sin(πt/12) + 10t = 50

4. Use numerical methods or graphing

5. Solution: t ≈ 3.2 months

Problem-Solving Framework

1. Understand

Read the problem carefully and identify what's given and what's asked.

2. Plan

Choose appropriate functions and set up equations or inequalities.

3. Solve

Execute your plan using algebraic or graphical methods.

4. Check

Verify your answer makes sense in the real-world context.
