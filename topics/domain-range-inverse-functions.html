Topic 1: Understanding Inverse Functions

Learn what inverse functions are, how they relate to original functions, and the fundamental relationship between their domains and ranges.

00:00 Study Time

Learning Objectives

  • Define inverse function and its properties
  • Understand the relationship between function and its inverse
  • Identify one-to-one functions that have inverses
  • Analyze the domain-range relationship in inverse functions

What are Inverse Functions?

Definition

An inverse function is a function that "undoes" the action of another function. Think of it like a mathematical "undo" button! If f is a function, then its inverse function f⁻¹ satisfies the property that f⁻¹(f(x)) = x and f(f⁻¹(x)) = x for all x in their respective domains.

f⁻¹(f(x)) = x and f(f⁻¹(x)) = x

where f⁻¹ is the inverse function of f

💡 Real-World Analogy: If f represents "putting on shoes", then f⁻¹ represents "taking off shoes". When you put on shoes and then take them off, you're back to where you started!

Key Insight:

The inverse function essentially reverses the mapping. If f maps x to y, then f⁻¹ maps y back to x. This creates a perfect "round trip" where applying both functions in sequence returns you to your starting point.

Key Properties of Inverse Functions

Domain-Range Swap

The domain of f becomes the range of f⁻¹, and the range of f becomes the domain of f⁻¹. This is the most important relationship for understanding inverse functions!

Example: If f has domain {1, 2, 3} and range {4, 5, 6}, then f⁻¹ has domain {4, 5, 6} and range {1, 2, 3}.

Graph Reflection

The graph of f⁻¹ is the reflection of the graph of f across the line y = x. This visual property helps us understand the relationship geometrically.

Visual Tip: Imagine folding the graph along the line y = x - the function and its inverse will overlap perfectly!

Why This Matters

Understanding these properties helps you:

  • Find the domain and range of inverse functions without calculating them
  • Verify if a function has an inverse
  • Understand the relationship between a function and its inverse visually
  • Solve real-world problems involving inverse relationships

Conditions for Inverse Functions

One-to-One Function
Required for inverse

A function must be one-to-one (injective) to have an inverse. This means each output corresponds to exactly one input.

Horizontal Line Test
Verification method

If any horizontal line intersects the graph of a function at most once, then the function is one-to-one and has an inverse.

Worked Examples

Example 1: Linear Function

Find the inverse of f(x) = 2x + 3 and determine the relationship between their domains and ranges.

Solution:

Step 1: Replace f(x) with y: y = 2x + 3

Step 2: Swap x and y: x = 2y + 3

Step 3: Solve for y: y = (x - 3)/2

Therefore: f⁻¹(x) = (x - 3)/2

Domain of f: All real numbers

Range of f: All real numbers

Domain of f⁻¹: All real numbers (same as range of f)

Range of f⁻¹: All real numbers (same as domain of f)

Example 2: Quadratic Function (Restricted)

Find the inverse of f(x) = x² for x ≥ 0 and analyze the domain-range relationship.

Solution:

Step 1: Replace f(x) with y: y = x², x ≥ 0

Step 2: Swap x and y: x = y², y ≥ 0

Step 3: Solve for y: y = √x (taking the positive square root since y ≥ 0)

Therefore: f⁻¹(x) = √x

Domain of f: x ≥ 0

Range of f: y ≥ 0

Domain of f⁻¹: x ≥ 0 (same as range of f)

Range of f⁻¹: y ≥ 0 (same as domain of f)

💡 Note: Without the restriction x ≥ 0, f(x) = x² would not be one-to-one (since both 2 and -2 map to 4), so it wouldn't have an inverse!

Example 3: Rational Function

Find the inverse of f(x) = 1/x and determine the domain-range relationship.

Solution:

Step 1: Replace f(x) with y: y = 1/x

Step 2: Swap x and y: x = 1/y

Step 3: Solve for y: y = 1/x

Therefore: f⁻¹(x) = 1/x (the function is its own inverse!)

Domain of f: x ≠ 0

Range of f: y ≠ 0

Domain of f⁻¹: x ≠ 0 (same as range of f)

Range of f⁻¹: y ≠ 0 (same as domain of f)

✨ Interesting: This function is symmetric - it's its own inverse! This happens because f(f(x)) = f(1/x) = 1/(1/x) = x.

Finding Domain of Inverse Functions

Fundamental Relationship

The domain of f⁻¹ is equal to the range of f. This is because the inverse function takes outputs from the original function as its inputs. Think of it this way: if f produces certain outputs, those outputs become the inputs for f⁻¹!

Domain of f⁻¹ = Range of f

This relationship is fundamental to understanding inverse functions

💡 Why This Works: Since f⁻¹ "undoes" f, it needs to accept all the values that f can produce. Therefore, the domain of f⁻¹ must include all values in the range of f!

Step-by-Step Process:
  1. Find the range of the original function f
  2. That range becomes the domain of f⁻¹
  3. Apply any additional restrictions if needed
Important Note:

Always check if the original function is one-to-one. If not, you may need to restrict its domain first before finding the inverse!

Step-by-Step Process

1 Find the Range of Original Function

Determine all possible output values of the original function f(x).

2 Apply Domain-Range Relationship

The domain of f⁻¹ is exactly the range of f.

3 Consider Restrictions

Apply any additional restrictions based on the context or problem requirements.

Common Domain Patterns

f(x) = 2x + 3
Domain of f⁻¹: All real numbers
f(x) = x², x ≥ 0
Domain of f⁻¹: x ≥ 0
f(x) = 1/x
Domain of f⁻¹: x ≠ 0

For f(x) = 3x - 1 with domain all real numbers, find the domain of f⁻¹(x) = (x + 1)/3.

Step 1: Find the range of f(x) = 3x - 1

Range of f: y ≠ 0 (all real numbers except 0)

Step 2: Domain of f⁻¹ = Range of f

Domain of f⁻¹: x ≠ 0

Finding Range of Inverse Functions

Fundamental Relationship

The range of f⁻¹ is equal to the domain of f. This is because the inverse function produces outputs that correspond to the inputs of the original function. Since f⁻¹ "undoes" f, it must produce all the values that f originally accepted as inputs!

Range of f⁻¹ = Domain of f

This relationship is the complement to the domain-range relationship

💡 Understanding: If f accepts inputs from a certain set (its domain), then f⁻¹ must produce outputs from that same set (its range). This creates perfect symmetry!

Step-by-Step Process:
  1. Identify the domain of the original function f
  2. That domain becomes the range of f⁻¹
  3. Verify with function analysis if needed
Connection:

Together with "Domain of f⁻¹ = Range of f", this completes the inverse relationship. Both relationships work together!

Step-by-Step Process

1 Identify the Domain of Original Function

Determine all possible input values of the original function f(x).

2 Apply Range-Domain Relationship

The range of f⁻¹ is exactly the domain of f.

3 Verify with Function Analysis

Confirm the range by analyzing the behavior of the inverse function.

Common Range Patterns

f(x) = 2x + 3
Range of f⁻¹: All real numbers
f(x) = x², x ≥ 0
Range of f⁻¹: y ≥ 0
f(x) = 1/x, x > 0
Range of f⁻¹: y > 0

Real-World Applications

Physics Applications

Temperature Conversion

Converting between Celsius and Fahrenheit: F = (9/5)C + 32 and C = (5/9)(F - 32)

Why This Works: These are inverse functions! Applying F(C) then C(F) returns the original temperature.

Example: 25°C → F(25) = 77°F → C(77) = 25°C ✓

Domain of F(C): All real numbers (any Celsius temperature)

Range of F(C): All real numbers (any Fahrenheit temperature)

Domain of C(F): All real numbers (same as range of F)

Range of C(F): All real numbers (same as domain of F)

🌍 Real-World Use: Weather forecasts, cooking recipes, scientific experiments, and international travel!

Kinetic Energy and Velocity

KE = (1/2)mv² and v = √(2KE/m) for velocity from kinetic energy

Understanding: Kinetic energy increases quadratically with velocity. The inverse function helps find velocity when energy is known.

Example: A 2kg object with 100J energy → v = √(2×100/2) = √100 = 10 m/s

Domain of KE(v): v ≥ 0 (non-negative velocities)

Range of KE(v): KE ≥ 0 (non-negative kinetic energy)

Domain of v(KE): KE ≥ 0 (same as range of KE)

Range of v(KE): v ≥ 0 (same as domain of KE)

⚡ Applications: Vehicle safety (crash tests), sports science, engineering design, and energy calculations!

Economics Applications

Supply and Demand

Price as a function of quantity: P = f(Q) and quantity as a function of price: Q = f⁻¹(P)

Economic Insight: Businesses use inverse functions to determine production quantities based on market prices. If price increases, they can calculate how much more to produce!

Example: If P = 2Q + 10, then Q = (P - 10)/2. At price $50, quantity = (50-10)/2 = 20 units.

Domain of P(Q): Q ≥ 0 (non-negative quantities)

Range of P(Q): P ≥ minimum price (usually ≥ 0)

Domain of Q(P): P ≥ minimum price (same as range of P)

Range of Q(P): Q ≥ 0 (same as domain of P)

💰 Business Application: Helps companies optimize production, set prices, and respond to market changes!

Interest Rate Calculations

Compound interest: A = P(1 + r)ᵗ and solving for rate: r = (A/P)^(1/t) - 1

Financial Application: Investors use this to compare different investment options and find equivalent interest rates!

Example: $1000 becomes $1500 in 5 years → r = (1500/1000)^(1/5) - 1 = 1.5^0.2 - 1 ≈ 8.45%

Domain of A(r): r ≥ -1, t > 0, P > 0

Range of A(r): A > 0 (when P > 0, r > -1)

Domain of r(A): A > 0, P > 0, t > 0, A ≥ P

Range of r(A): r ≥ -1 (can be negative for losses)

💵 Real-World Use: Comparing savings accounts, loans, investments, and retirement planning!

Engineering Applications

Signal Processing

Frequency and wavelength relationship: f = c/λ and λ = c/f

Engineering Application: Essential for designing antennas, radio systems, and optical devices. The inverse relationship helps engineers choose the right frequency for a desired wavelength!

Example: Radio wave with f = 100 MHz, c = 3×10⁸ m/s → λ = 3×10⁸ / 10⁸ = 3 meters

Domain of f(λ): λ > 0 (positive wavelengths)

Range of f(λ): f > 0 (positive frequencies)

Domain of λ(f): f > 0 (same as range of f)

Range of λ(f): λ > 0 (same as domain of f)

📡 Real-World Use: Radio broadcasting, WiFi design, satellite communication, and medical imaging (MRI, X-rays)!

Vehicle Performance

Speed and fuel consumption: C = f(v) and optimal speed: v = f⁻¹(min C)

Automotive Engineering: Manufacturers use inverse functions to optimize fuel efficiency. Finding the speed that minimizes consumption helps design more efficient vehicles!

Example: For a simplified model C = 50/v, optimal speed occurs at v = √(50/C). At C = 8 L/100km, optimal v ≈ 2.5 km/h (simplified model).

Domain of C(v): v > 0 (positive speeds)

Range of C(v): C > 0 (positive fuel consumption)

Domain of v(C): C > 0 (same as range of C)

Range of v(C): v > 0 (same as domain of C)

🚗 Practical Impact: Helps drivers save fuel, reduces emissions, and guides vehicle design for better efficiency!

Worked Examples

Example 1: Temperature Conversion

Convert 25°C to Fahrenheit using the inverse function relationship. The conversion formula is F = (9/5)C + 32.

Solution:

Using F(C) = (9/5)C + 32:

F(25) = (9/5)(25) + 32 = 45 + 32 = 77°F

Domain Analysis:

• Domain of F(C): All real numbers (any Celsius temperature)

• Range of F(C): All real numbers (any Fahrenheit temperature)

Inverse Function C(F) = (5/9)(F - 32):

• Domain of C(F): All real numbers (same as range of F)

Range of C(F): All real numbers (same as domain of F)

🌍 Real-World Application: This inverse relationship allows meteorologists to convert temperatures in either direction seamlessly!

Example 2: Kinetic Energy Problem

A car with mass 1500 kg is moving at 20 m/s. Find its kinetic energy, then determine the velocity if the kinetic energy is 300,000 J.

Solution:

Part 1: KE = (1/2)mv² = (1/2)(1500)(20)² = (1/2)(1500)(400) = 300,000 J

Part 2: Using inverse function v = √(2KE/m):

v = √(2 × 300,000 / 1500) = √(600,000 / 1500) = √400 = 20 m/s

Domain-Range Analysis:

• Domain of KE(v): v ≥ 0 (non-negative velocities)

• Range of KE(v): KE ≥ 0 (non-negative kinetic energy)

• Domain of v(KE): KE ≥ 0 (same as range of KE)

Range of v(KE): v ≥ 0 (same as domain of KE)

⚡ Physics Insight: The inverse relationship allows engineers to calculate required velocities for specific energy levels!

Example 3: Supply and Demand

A supply function is given by P = 2Q + 10, where P is price and Q is quantity. Find the quantity supplied when price is $50, and analyze the domain-range relationship.

Solution:

Using inverse function Q = (P - 10)/2:

Q(50) = (50 - 10)/2 = 40/2 = 20 units

Domain-Range Analysis:

• Domain of P(Q): Q ≥ 0 (non-negative quantities)

• Range of P(Q): P ≥ 10 (prices start at $10)

• Domain of Q(P): P ≥ 10 (same as range of P)

Range of Q(P): Q ≥ 0 (same as domain of P)

💰 Economic Application: This inverse relationship helps businesses determine production quantities based on market prices!
