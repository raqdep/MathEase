Addition and Subtraction of Functions: Complete Guide

Addition of Functions

To add two functions f(x) and g(x), we add their outputs for the same input value.

Addition Formula:

(f + g)(x) = f(x) + g(x)

Read as: "f plus g of x equals f of x plus g of x"

Step-by-Step Process:
  1. 1. Write the addition formula
  2. 2. Substitute the function expressions
  3. 3. Combine like terms
  4. 4. Simplify the result

Subtraction of Functions

To subtract two functions f(x) and g(x), we subtract their outputs for the same input value.

Subtraction Formula:

(f - g)(x) = f(x) - g(x)

Read as: "f minus g of x equals f of x minus g of x"

Step-by-Step Process:
  1. 1. Write the subtraction formula
  2. 2. Substitute the function expressions
  3. 3. Distribute the negative sign
  4. 4. Combine like terms

Step-by-Step Examples

Addition Example

Given functions:

f(x) = 3x - 2

g(x) = x² + 1

Find: (f + g)(x)

1

Write the addition formula

(f + g)(x) = f(x) + g(x)
2

Substitute the functions

(f + g)(x) = (3x - 2) + (x² + 1)
3

Remove parentheses

(f + g)(x) = 3x - 2 + x² + 1
4

Combine like terms

(f + g)(x) = x² + 3x - 1
Subtraction Example

Given functions:

f(x) = x² + 2

g(x) = x + 1

Find: (f - g)(x)

1

Write the subtraction formula

(f - g)(x) = f(x) - g(x)
2

Substitute the functions

(f - g)(x) = (x² + 2) - (x + 1)
3

Distribute the negative sign

(f - g)(x) = x² + 2 - x - 1
4

Combine like terms

(f - g)(x) = x² - x + 1

Important: Domain of Combined Functions

Key Rule:

The domain of (f + g) and (f - g) is the intersection of the domains of f and g.

This means both functions must be defined for the same input values.

Example: Domain Intersection

f(x) = √x (domain: x ≥ 0)

g(x) = 1/x (domain: x ≠ 0)

Domain of (f + g): x > 0

Example: All Real Numbers

f(x) = x² (domain: all real numbers)

g(x) = 2x + 1 (domain: all real numbers)

Domain of (f + g): all real numbers

Multiplication of Functions: Complete Guide

Multiplication of Functions

To multiply two functions f(x) and g(x), we multiply their outputs for the same input value.

Multiplication Formula:

(f • g)(x) = f(x) • g(x)

Read as: "f times g of x equals f of x times g of x"

Step-by-Step Process:
  1. 1. Write the multiplication formula
  2. 2. Substitute the function expressions
  3. 3. Apply distributive property
  4. 4. Combine like terms

Step-by-Step Examples

Multiplication Example

Given functions:

f(x) = 2x + 1

g(x) = x - 3

Find: (f • g)(x)

1

Write the multiplication formula

(f • g)(x) = f(x) • g(x)
2

Substitute the functions

(f • g)(x) = (2x + 1)(x - 3)
3

Apply distributive property (FOIL method)

(f • g)(x) = 2x(x) + 2x(-3) + 1(x) + 1(-3)
4

Simplify each term

(f • g)(x) = 2x² - 6x + x - 3
5

Combine like terms

(f • g)(x) = 2x² - 5x - 3

FOIL Method for Multiplying Binomials

FOIL stands for:
  • First: Multiply the first terms
  • Outer: Multiply the outer terms
  • Inner: Multiply the inner terms
  • Last: Multiply the last terms
Example: (2x + 1)(x - 3)

F: 2x • x = 2x²

O: 2x • (-3) = -6x

I: 1 • x = x

L: 1 • (-3) = -3

Result: 2x² - 6x + x - 3 = 2x² - 5x - 3

Practice 1: Linear × Linear

Given: f(x) = 3x + 2, g(x) = x - 1

Find: (f • g)(x)

(f • g)(x) = (3x + 2)(x - 1) = 3x² - 3x + 2x - 2 = 3x² - x - 2

Practice 2: Quadratic × Linear

Given: f(x) = x² + 1, g(x) = 2x - 1

Find: (f • g)(x)

(f • g)(x) = (x² + 1)(2x - 1) = 2x³ - x² + 2x - 1

Division of Functions: Complete Guide

Division of Functions

To divide two functions f(x) and g(x), we divide their outputs for the same input value, but g(x) cannot be zero.

Division Formula:

(f / g)(x) = f(x) / g(x), where g(x) ≠ 0

Read as: "f divided by g of x equals f of x divided by g of x"

Step-by-Step Process:
  1. 1. Write the division formula
  2. 2. Substitute the function expressions
  3. 3. Factor if possible
  4. 4. Simplify and identify restrictions

Important: Domain Restrictions

Key Rules:
  • • The domain of (f / g) is the intersection of the domains of f and g
  • • EXCEPT where g(x) = 0
  • • These excluded values are called vertical asymptotes
  • • Always check for common factors that can be canceled

Step-by-Step Examples

Division Example

Given functions:

f(x) = x² - 4

g(x) = x - 2

Find: (f / g)(x)

1

Write the division formula

(f / g)(x) = f(x) / g(x)
2

Substitute the functions

(f / g)(x) = (x² - 4) / (x - 2)
3

Factor the numerator (difference of squares)

(f / g)(x) = (x - 2)(x + 2) / (x - 2)
4

Cancel common factors

(f / g)(x) = x + 2, where x ≠ 2

Properties of Rational Functions

Domain Restrictions
  • • Find where denominator = 0
  • • These are excluded from domain
  • • Create vertical asymptotes
Simplification
  • • Factor numerator and denominator
  • • Cancel common factors
  • • Note any restrictions

Given: f(x) = x² - 9, g(x) = x + 3

Find: (f / g)(x)

(f / g)(x) = (x² - 9) / (x + 3) = (x - 3)(x + 3) / (x + 3) = x - 3, where x ≠ -3

Practice 2: Complex Division

Given: f(x) = x² + 5x + 6, g(x) = x + 2

Find: (f / g)(x)

(f / g)(x) = (x² + 5x + 6) / (x + 2) = (x + 2)(x + 3) / (x + 2) = x + 3, where x ≠ -2

Composition of Functions: Complete Guide

What is Function Composition?

Function composition is like a "function of a function." The output of one function becomes the input of another function.

Composition Formula:

(f ∘ g)(x) = f(g(x))

Read as: "f composed with g of x equals f of g of x"

Step-by-Step Process:
  1. 1. Start with the inner function g(x)
  2. 2. Substitute g(x) into f(x)
  3. 3. Simplify the result
  4. 4. Determine the domain

Visual Understanding

x

Input

g(x)

Inner Function

f(g(x))

Final Output

The composition (f ∘ g)(x) means: x → g(x) → f(g(x))

Step-by-Step Examples

Example 1: (f ∘ g)(x)

Given functions:

f(x) = 2x + 1

g(x) = x - 3

Find: (f ∘ g)(x)

1

Write the composition formula

(f ∘ g)(x) = f(g(x))
2

Substitute g(x) into f(x)

(f ∘ g)(x) = f(x - 3)
3

Apply f to the result

(f ∘ g)(x) = 2(x - 3) + 1
4

Simplify

(f ∘ g)(x) = 2x - 5
Example 2: (g ∘ f)(x)

Same functions:

f(x) = 2x + 1

g(x) = x - 3

Find: (g ∘ f)(x)

1

Write the composition formula

(g ∘ f)(x) = g(f(x))
2

Substitute f(x) into g(x)

(g ∘ f)(x) = g(2x + 1)
3

Apply g to the result

(g ∘ f)(x) = (2x + 1) - 3
4

Simplify

(g ∘ f)(x) = 2x - 2

Important: Order Matters!

Key Point: (f ∘ g)(x) ≠ (g ∘ f)(x) in general!

(f ∘ g)(x) = 2x - 5

f composed with g

(g ∘ f)(x) = 2x - 2

g composed with f

Notice how the results are different! The order of composition matters.

Real-World Applications of Function Operations

Economics: Profit and Revenue Functions

Problem:

A company produces widgets. The revenue function is R(x) = 50x - 0.5x² and the cost function is C(x) = 20x + 100, where x is the number of widgets produced.

Find the profit function P(x) and determine the maximum profit.

1

Define the profit function

P(x) = R(x) - C(x)
2

Substitute the functions

P(x) = (50x - 0.5x²) - (20x + 100)
3

Simplify

P(x) = -0.5x² + 30x - 100
4

Find maximum profit (vertex of parabola)

Maximum at x = 30, P(30) = 350

Physics: Motion and Composition

Problem:

A ball is thrown upward with velocity v(t) = 20 - 9.8t (m/s). The height function is h(v) = v²/19.6.

Find the height as a function of time h(t).

1

We need h(t) = h(v(t))

h(t) = h(v(t)) = h(20 - 9.8t)
2

Substitute v(t) into h(v)

h(t) = (20 - 9.8t)² / 19.6
3

Expand and simplify

h(t) = 20t - 4.9t²

Engineering: Circuit Analysis

Problem:

In an electrical circuit, the current I(t) = 2sin(t) and resistance R(t) = 5 + 0.1t.

Find the power function P(t) = I²(t) × R(t).

1

Write the power formula

P(t) = I²(t) × R(t)
2

Substitute the functions

P(t) = (2sin(t))² × (5 + 0.1t)
3

Simplify

P(t) = 4sin²(t) × (5 + 0.1t)

Practice Problems

Problem 1: Business

A store's daily revenue is R(x) = 100x - x² and costs are C(x) = 50x + 200, where x is items sold.

Find the profit function and break-even points.

P(x) = R(x) - C(x) = -x² + 50x - 200
Break-even: x = 5 and x = 40

Problem 2: Science

Temperature T(h) = 20 - 0.0065h and pressure P(T) = 101.3e^(-0.034T).

Find pressure as a function of height P(h).

P(h) = P(T(h)) = 101.3e^(-0.034(20 - 0.0065h))
= 101.3e^(-0.68 + 0.000221h)
