Topic 1: Understanding Domain

Learn what domain means for rational functions, how to identify restrictions, and why certain values are excluded from the domain.

00:00 Study Time

Learning Objectives

  • Define domain of a rational function
  • Identify domain restrictions from the denominator
  • Express domain in interval notation
  • Analyze domain restrictions in real-world contexts

What is Domain?

Definition

The domain of a rational function is the set of all real numbers for which the function is defined. For rational functions, this means all real numbers except those that make the denominator equal to zero.

Domain = {x ∈ ℝ | Q(x) ≠ 0}

where Q(x) is the denominator polynomial

Why Domain Restrictions Matter

Division by Zero

When the denominator equals zero, the function becomes undefined, creating vertical asymptotes or holes in the graph.

Graph Behavior

Domain restrictions determine where the function has vertical asymptotes and affect the overall shape of the graph.

Common Domain Patterns

f(x) = 1/x
Domain: x ≠ 0
f(x) = 1/(x-2)
Domain: x ≠ 2
f(x) = (x+1)/(x²-4)
Domain: x ≠ ±2

Concept Deep Dive: Understanding Domain

1 The Foundation: Why Domain Matters

The domain is fundamental because it tells us where a function exists. For rational functions, the domain is determined by one critical rule: the denominator must never equal zero.

Key Insight: When you divide by zero, mathematics breaks down. That's why we exclude values that make the denominator zero from the domain.

2 Step-by-Step Process for Finding Domain

A
Identify the Denominator:

Look at the bottom part of the fraction. This is what you need to analyze.

B
Set Denominator = 0:

Solve this equation to find the "forbidden" values.

C
Exclude Solutions:

Remove these values from all real numbers to get the domain.

D
Express in Interval Notation:

Write your answer using intervals, which is the standard mathematical notation.

3 Common Patterns and Their Domains

Function Type Example Domain
Simple Reciprocal f(x) = 1/x x ≠ 0
Shifted Reciprocal f(x) = 1/(x-2) x ≠ 2
Quadratic Denominator f(x) = 1/(x²-4) x ≠ ±2
Linear over Linear f(x) = (x+1)/(x-3) x ≠ 3

4 Tips and Tricks

✓ Always Factor First

If the denominator is a polynomial, factor it completely before setting it equal to zero.

✓ Check All Factors

When you factor, make sure to check each factor separately. Each factor gives you a restriction.

✓ Use Interval Notation

Interval notation is cleaner and more professional than writing "all real numbers except..."

⚠ Watch for Holes

Sometimes factors cancel out, creating "holes" instead of vertical asymptotes. Always check!

More Worked Examples

1 Example 1: Simple Domain

Find the domain of f(x) = 1/(x-3)

📝 Step-by-Step Solution:

  1. Identify the denominator: The denominator is (x - 3)
  2. Set denominator equal to zero: x - 3 = 0
  3. Solve for x: x = 3
  4. Exclude from domain: Since x = 3 makes the denominator zero, we exclude it
  5. Final answer: Domain is all real numbers except x = 3

✅ Domain:

(-∞, 3) ∪ (3, ∞)

This means: all numbers less than 3, combined with all numbers greater than 3

2 Example 2: Multiple Restrictions

Find the domain of f(x) = (x+2)/(x²-9)

📝 Step-by-Step Solution:

  1. Identify the denominator: The denominator is (x² - 9)
  2. Set denominator equal to zero: x² - 9 = 0
  3. Factor the quadratic: (x + 3)(x - 3) = 0 (difference of squares)
  4. Apply zero product property: Either (x + 3) = 0 or (x - 3) = 0
  5. Solve both equations: x = -3 or x = 3
  6. Exclude both values: Domain excludes x = -3 and x = 3
  7. Final answer: Domain is all real numbers except x = -3 and x = 3

✅ Domain:

(-∞, -3) ∪ (-3, 3) ∪ (3, ∞)

This creates three intervals: numbers less than -3, between -3 and 3, and greater than 3

3 Example 3: Factoring Required

Find the domain of f(x) = (x+1)/(x²-5x+6)

📝 Step-by-Step Solution:

  1. Identify the denominator: The denominator is (x² - 5x + 6)
  2. Set denominator equal to zero: x² - 5x + 6 = 0
  3. Factor the quadratic: Find two numbers that multiply to 6 and add to -5
  4. Factored form: (x - 2)(x - 3) = 0
  5. Solve: x = 2 or x = 3
  6. Exclude both values: Domain excludes x = 2 and x = 3

✅ Domain:

(-∞, 2) ∪ (2, 3) ∪ (3, ∞)

4 Example 4: Perfect Square Denominator

Find the domain of f(x) = (2x+1)/(x²-4x+4)

📝 Step-by-Step Solution:

  1. Identify the denominator: The denominator is (x² - 4x + 4)
  2. Recognize the pattern: This is a perfect square trinomial!
  3. Factor: (x - 2)² = 0
  4. Solve: x - 2 = 0, so x = 2
  5. Note: Even though it's squared, we only exclude x = 2 once

✅ Domain:

(-∞, 2) ∪ (2, ∞)

💡 Tip: Perfect squares still give only one restriction, not two!

Topic 2: Understanding Range

Learn what range means for rational functions, how to determine the set of possible output values, and the relationship between range and horizontal asymptotes.

What is Range?

Definition

The range of a rational function is the set of all possible output values (y-values) that the function can produce. Unlike domain, range is often more complex to determine and depends on the behavior of the function.

Range = {y ∈ ℝ | y = f(x) for some x in domain}

where f(x) is the rational function

Factors Affecting Range

1 Horizontal Asymptotes

Horizontal asymptotes limit the range by providing boundaries that the function approaches but never crosses.

2 Vertical Asymptotes

Vertical asymptotes create gaps in the domain but don't directly restrict the range.

3 Function Behavior

The overall behavior of the function, including its end behavior, determines the complete range.

Common Range Patterns

f(x) = 1/x
Range: y ≠ 0
f(x) = (2x+1)/(3x-2)
Range: y ≠ 2/3
f(x) = (x²-1)/(x+2)
Range: All real numbers

Concept Deep Dive: Understanding Range

1 The Challenge: Range is More Complex

Unlike domain (which focuses on input restrictions), range focuses on output possibilities. The range tells us what y-values the function can actually produce. This is trickier because we need to understand the function's behavior.

Key Insight: Horizontal asymptotes are your best friend when finding range! They show you the boundaries that the function approaches but may never cross.

2 Horizontal Asymptote Rules

Degree Comparison Horizontal Asymptote Range Impact
deg(num) < deg(den) y = 0 Range may exclude 0
deg(num) = deg(den) y = a/b Range excludes asymptote value
deg(num) > deg(den) None Range is all real numbers

3 Step-by-Step Process for Finding Range

A
Find Horizontal Asymptote:

Compare degrees of numerator and denominator. If equal, divide leading coefficients.

B
Check if Asymptote is Attainable:

Solve f(x) = asymptote. If no solution exists, exclude that value from range.

C
Analyze Function Behavior:

Consider end behavior and any local extrema that might restrict the range.

D
Express in Interval Notation:

Write your answer using proper interval notation, excluding any unattainable values.

Real-World Applications

Physics Applications

Electrical Circuits

In parallel circuits, the total resistance is given by 1/R_total = 1/R₁ + 1/R₂ + ... + 1/Rₙ

Domain: All positive real numbers (resistance values)

Range: All positive real numbers (total resistance)

Optics - Lens Equation

The lens equation 1/f = 1/d₀ + 1/dᵢ relates focal length to object and image distances.

Domain: All positive real numbers (distances)

Range: All positive real numbers (focal length)

Economics Applications

Average Cost Function

AC(x) = C(x)/x where C(x) is total cost and x is quantity produced.

Domain: x > 0 (positive production quantities)

Range: All positive real numbers (average cost per unit)

Market Share

Market share as a function of advertising expenditure: S(a) = ka/(a + b)

Domain: a ≥ 0 (non-negative advertising expenditure)

Range: 0 ≤ S(a) < k (market share between 0 and maximum k)

Engineering Applications

Vehicle Performance

Fuel efficiency as a function of speed: E(v) = av/(v² + bv + c)

Domain: v > 0 (positive speeds)

Range: 0 < E(v) < a/c (efficiency between 0 and maximum)

Control Systems

Transfer functions in control systems: G(s) = N(s)/D(s)

Domain: Complex plane excluding poles

Range: Complex plane (transfer function values)
